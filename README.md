### Deploying the Solution

1. cd into either the `blueprints` or `custom-constructs` folder.
2. run `npm i`.
3. run `npm run deploy:develop`.

### Removing the Solution

To remove the solution run the following:

1. In the `blueprints` or `custom-constructs` folder run the following `npm run remove:develop`.
